About This Blog

Simple Blog Application

This is a simple blog web application built with Flask and deployed on AWS EC2. It allows users to create, read, edit, and delete blog posts.6

Features:
  • Create new blog posts
  • View all posts on the homepage
  • Read individual posts
  • Edit existing posts
  • Delete posts
  • Responsive design with Bootstrap
Technology Stack:
  • Backend: Python Flask
  • Database: SQLite
  • Frontend: Bootstrap 5
  • Deployment: AWS EC2
